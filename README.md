# 饺满灶台 (jmzt) 项目目录指南

本目录集成了“饺满灶台”相关的所有核心组件与资源，包含前端管理后台、后端服务及数据库备份。

## 📁 目录结构说明

* **`my-ecommerce-admin/`**: 电商管理后台（前端项目，基于 Vite + Vue3 构建）。
* **`myxcxbkg/`**: 后端核心逻辑与服务组件。
* **`pay1/`**: 支付接口处理模块。
* **`go/`**: Go 语言相关后端代码或运行环境。
* **`gin_vue_admin_xxx.sql`**: 数据库初始备份文件。

## 🚀 快速开始

1. **前端启动**: 
   - 进入 `my-ecommerce-admin`
   - 执行 `npm install` 补齐依赖（建议使用 vuedraggable@4.1.0）
   - 执行 `npm run dev` 启动。
2. **后端配置**:
   - 检查各模块下的 `.env` 文件，确保 API 地址指向正确。

## ⚠️ 注意事项
- Linux 环境下请注意文件名大小写。
- 遇到页面点不开的情况，请务必检查浏览器 F12 控制台报错。
