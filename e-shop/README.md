# 喵购商城

自己研究用uni-app-x开发的电商项目，会不定期更新内容，大家感兴趣可以自己下载研究一下。

## 说明
1. css部分使用了uni.scss部分变量, 其他语法已剔除特殊语法, 除变量外可直接替换其他css预处理器
2. app.uvue下定义了个别全局样式
3. 该项目为效果展示学习使用, 数据和图片来自网络，均只用于案例展示，可以删除，请勿商用！
